# Augmented Data
Augmented training data for Hausa and Kinyarwanda is generated by running `notebooks/05_augmentation.ipynb` on Kaggle.

## Files generated
- `hau_augmented.csv` — 1,081 pairs (62.3% survival rate at quality threshold 0.30)
- `kin_augmented.csv` — 777 pairs (99.9% survival rate at quality threshold 0.30)

## Notes
- Files are not committed to GitHub due to file size constraints. Run the augmentation notebook on Kaggle with GPU acceleration to regenerate. The original training data used for augmentation is in `data/raw/hau/train.csv` and `data/raw/kin/train.csv`.
- Back-translation was performed using Helsinki-NLP/opus-mt-mul-en and opus-mt-en-mul. Quality filtering used cosine similarity between original and back-translated sentence embeddings.

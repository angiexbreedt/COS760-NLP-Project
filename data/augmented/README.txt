# Augmented Data
 
Augmented training data for Hausa and Kinyarwanda generated by `notebooks/05-augmentation.ipynb`.
 
## Files generated
 
- `hau_augmented.csv` — 1,081 pairs (62.3% survival rate at quality threshold 0.30)
- `kin_augmented.csv` — 777 pairs (99.9% survival rate at quality threshold 0.30)
## Notes
 
Files are not committed to GitHub due to file size. Run the augmentation notebook on Kaggle with GPU T4 to regenerate.
 
**Method:** Back-translation using `Helsinki-NLP/opus-mt-mul-en` and `opus-mt-en-mul`. Quality filtering used cosine similarity (threshold 0.30) between original and back-translated sentence embeddings using `paraphrase-multilingual-mpnet-base-v2`.
 
**Hausa note:** At the standard threshold of 0.75 only 0.2% of pairs survived, confirming limited Hausa translation resource maturity. Threshold was lowered to 0.30 to obtain a usable augmented set.
